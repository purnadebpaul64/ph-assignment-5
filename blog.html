Question-1: What are the different ways to select an element in the DOM?

There are few ways to select an element in the dom some of them are getElementById(), getElementsByClassName(),getElementsByTagName(), querySelector() etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML returns or sets the HTML code + Text inside the element on the other hand innerText Returns or sets the only text inside the element and textContent returns or sets the Full Text with hidden text inside the element

Question-3: What is event delegation in the DOM?

Event delegation is a way which allows the parent to listen to events triggered by its child elements using Event Bubbling

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept where an event starts from the target element(child) and travels upward to its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element using document.createElement("tagName") and we can add elements in dom by using appendChild(), innerHTML, append() etc and we can remove an element by using remove() or removeChild()
